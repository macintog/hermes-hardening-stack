# Subagent dispatch card

Use this compact form if the harness needs a short task card.

```text
Workstream: <name>
Source of truth: connected GitHub repo macintog/hermes-hardening-stack and applied Hermes patch stack.
Goal: <one sentence>
Read first: <files>
Do: <tasks>
Do not: rely on regex/fences as security boundary; drop patch 0005; weaken tests.
Output: findings, file-level changes, tests, residual risks.
```
